# Lead Qualification Web Agent

## Overview
This project is a demo web agent that identifies, enriches, and ranks scientific and business
profiles based on their likelihood of adopting 3D in-vitro models for therapy development.

The focus is on business logic, probability scoring, and dashboard output rather than
full-scale web scraping.

## Run Instructions
pip install -r requirements.txt
streamlit run app/app.py
